\# Project Charter



\## Project Name



Operations Process Optimisation



\## Project Goal



Modernize the operations platform by implementing Agile Scrum practices to improve compliance, payment processing, and customer self-service capabilities.



\## Objectives



\- Improve regulatory compliance

\- Modernize payment processing

\- Enhance customer self-service

\- Deliver functionality using Scrum



\## Scope



\### In Scope



\- Compliance improvements

\- Payment Gateway modernization

\- Customer Portal enhancements



\### Out of Scope



\- Mobile application

\- Legacy infrastructure replacement



\## Success Criteria



\- Four planned sprints

\- Prioritized backlog

\- Estimated Story Points

\- Progress tracked using Jira reports
