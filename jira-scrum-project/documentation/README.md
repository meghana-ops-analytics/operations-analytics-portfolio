# Jira Scrum Project

## Overview

This project demonstrates the setup and execution of a Scrum project using Jira Cloud.

The project simulates the modernization of an Operations Process Optimisation platform through Agile Scrum practices.

## Objectives

- Create a Scrum project in Jira
- Define Epics and User Stories
- Plan multiple sprints
- Estimate Story Points
- Track work using Scrum Board
- Monitor progress using Jira reports

## Project Structure

- 3 Epics
- 28 User Stories
- 4 Planned Sprints
- Story Point Estimation
- Scrum Board Workflow
- Burndown Chart
- Burnup Chart
- Cumulative Flow Diagram
